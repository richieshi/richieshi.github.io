Projects

YouTube Music Player

This is a desktop application that you can use to create playlists of and listen to music that is available on YouTube. The front end was created using React, Redux and Electron, and the backend was created using Python (Flask library) and MySQL.

Yelp Review Classifier

I trained a classifier using a Yelp dataset of reviews to categorize reviews by their sentiment (i.e. bad, neutral, good). I created this in order to better understand the process of using machine learning algorithms to learn from data as well as research and apply NLP techniques to understand text. Currently, the classifier successfully predicts the sentiment of reviews with an accuracy of 84% using a logistic regression model.

Straights

Straights is a card game where you try to empty your hand by creating straights on the board. You can play against other players or against the computer. This game was created using C++ with the GTK+ library for the user interface. This was created for a software design principles class so source code will not be available.
